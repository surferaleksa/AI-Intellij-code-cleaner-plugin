
AI Code Clarity Tool

An IntelliJ plugin that uses AI to clean up and refactor code to improve readability, structure, and maintainability without changing functionality.

Setup:
1. Get a free API key from [console.groq.com](https://console.groq.com)
2. Open src/main/kotlin/MyPluginAction.kt
3. Replace `YOUR_GROQ_API_KEY_HERE` with your actual key
4. Run the plugin in gradle with runIde

Features:

* AI-assisted code cleanup
* Readability-focused refactoring
* Cleaner formatting and structure
* IntelliJ IDEA integration
